1. Lister les hôtes du groupe nodes

Affiche les hôtes du groupe nodes en utilisant l'inventaire configuré dans inventory/hosts.yml :

ansible nodes --list-hosts

2. Tester la connectivité SSH avec tous les nœuds

Envoie une requête ping (module ping) pour vérifier la connectivité SSH :

ansible nodes -m ping

3. Vérifier les variables spécifiques à un nœud

Pour afficher les variables associées à node4 :

ansible node4 -m debug -a "var=hostvars[inventory_hostname]"

4. Installer les paquets communs sur tous les nœuds

Installe les paquets définis dans group_vars/nodes.yml :

ansible nodes -m apt -a "name={{ item }} state=present" --extra-vars "item={{ packages }}" --become

5. Installer des paquets supplémentaires sur un nœud spécifique

Installe les paquets spécifiques à node4 définis dans host_vars/node4.yml :

ansible node4 -m apt -a "name={{ item }} state=present" --extra-vars "item={{ additional_packages }}" --become

6. Configurer le fuseau horaire d’un nœud

Met à jour le fuseau horaire d’un nœud en utilisant les variables définies :

ansible node4 -m shell -a "echo '{{ timezone }}' > /etc/timezone && dpkg-reconfigure -f noninteractive tzdata" --become

7. Configurer un serveur NTP pour la synchronisation du temps

Configure le serveur NTP spécifique à un nœud :

ansible node4 -m command -a "echo 'server {{ ntp_server }}' >> /etc/ntp.conf" --become

8. Activer un service sur un nœud

Pour activer un service défini dans les variables :

ansible node4 -m service -a "name=ntp state=started enabled=yes" --become

9. Appliquer les règles de pare-feu sur tous les nœuds

Ajoute des règles de pare-feu pour les ports définis dans group_vars/nodes.yml :

ansible nodes -m command -a "iptables -A INPUT -p tcp --dport {{ item.port }} -j ACCEPT" --extra-vars "item={{ firewall_rules }}" --become

10. Copier un fichier vers un nœud

Pour copier un fichier local vers un répertoire distant :

ansible node4 -m copy -a "src=/path/to/local/file dest=/etc/remote_file mode=0644" --become

11. Redémarrer un service

Redémarre le service SSH sur tous les nœuds :

ansible nodes -m service -a "name=ssh state=restarted" --become

12. Afficher la partition disque utilisée par un nœud

Utilise la variable disk_partition pour vérifier l’espace disque :

ansible node4 -m command -a "df -h {{ disk_partition }}" --become

13. Exécuter une commande spécifique sur tous les nœuds

Exécute une commande pour vérifier la version de Python installée :

ansible nodes -m command -a "python3 --version"

14. Lister les variables d’un groupe

Pour lister toutes les variables appliquées au groupe nodes :

ansible nodes -m debug -a "var=hostvars"
