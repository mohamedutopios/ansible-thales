Explications des variables :

  1. Groupe commun (group_vars/nodes.yml) :

    • packages : Liste des paquets à installer sur tous les nœuds.
    • ssh_port : Port SSH commun pour les nœuds.
    • firewall_rules : Règles de pare-feu communes.

  2. Variables spécifiques aux nœuds (host_vars/nodeX.yml) :

    • hostname : Nom unique pour chaque nœud.
    • disk_partition : Partition disque pour chaque nœud.
    • timezone : Fuseau horaire spécifique.
    • ntp_server : Serveur NTP à synchroniser.
    • additional_packages : Paquets additionnels spécifiques à chaque nœud.
    • services_to_enable : Services spécifiques à activer sur chaque nœud.

Cette configuration permettra de personnaliser chaque nœud tout en partageant des configurations communes entre eux via group_vars/nodes.yml.
